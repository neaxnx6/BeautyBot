# BeautyBot (SaaS MVP)

**Цель**: Telegram-бот для автоматической записи клиентов на услуги (Маникюр).
**Стек**: Python 3.10+, aiogram 3.x, SQLite (на старте).

## Архитектура
*   **Monolith Bot**: Один экземпляр бота обслуживает всех мастеров.
*   **Database**:
    *   `users`: Клиенты (id, name, phone).
    *   `masters`: Мастера (id, name, description).
    *   `slots`: Слоты времени (master_id, time, is_booked, client_id).

## Структура Проекта
*   `bot.py`: Точка входа.
*   `config.py`: Конфигурация (токены).
*   `database/`: Работа с БД.
*   `handlers/`:
    *   `client.py`: Логика записи (для клиентов).
    *   `master.py`: Логика управления расписанием (для мастера).
    *   `admin.py`: Техническая админка (для тебя).
